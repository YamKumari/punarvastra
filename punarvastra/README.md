# PunarVastra - Thrift Store

Java 25, JSP, Servlets, MySQL, Tomcat 10

## Setup
1. Start XAMPP MySQL
2. Import sql/schema.sql to phpMyAdmin
3. Run: mvn clean package cargo:run
4. Open: http://localhost:9090/punarvastra/
